Error Handling & Fallbacks
Profiler Fails
Dashboard agent falls back to current behavior — raw sample rows in prompt. Graceful degradation, never blocks.
Cache Miss on Refinement
Re-profile from state.queries[] execution rows. Slight delay but transparent to user.
Tile Data Not Cached
Client falls back to existing behavior — aggregates in-browser from raw rows. KpiSparklineCard keeps its computeKpiValue().
Large Dataset (>10K rows)
Profiler uses statistical sampling (1K rows) for shape detection. Aggregates computed via SQL (COUNT/SUM/AVG pushed to DB).
