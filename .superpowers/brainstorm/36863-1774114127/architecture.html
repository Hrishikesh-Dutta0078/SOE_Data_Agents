What the Data Profiler computes per source:

Column Analysis
  • Inferred type (numeric, date, categorical, text)
  • Cardinality (distinct count)
  • Null ratio per column
  • Value distribution (min/max/mean/median)
  • Top N distinct values (for low-cardinality)
Shape Detection
  • Time-series detection (ordered dates + measures)
  • Categorical grouping candidates
  • KPI candidates (single-value aggregatable)
  • High-cardinality warnings
  • Correlation hints between numeric columns
Anomaly Signals
  • Outlier detection (IQR method)
  • Trend direction (increasing/decreasing/flat)
  • Concentration (top 3 values = X% of data)
  • Period-over-period changes
  • Missing data patterns
Pre-computed Outputs
  • KPI aggregates (sum, avg, count per candidate)
  • Slicer distinct values (ready to serve)
  • Grouped aggregates for top chart candidates
  • Sparkline data for time-series KPIs
  • Cross-source dimension overlap map
