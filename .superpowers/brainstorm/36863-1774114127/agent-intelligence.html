Smart Chart Type Selection
Data Shape Detected Chart Type Why
Date column + numeric measures, ordered line or area Time-series → trend visualization
Categorical col (2-8 distinct) + one numeric pie Small set → part-of-whole
Categorical col (3-20 distinct) + numeric measures bar Medium cardinality → comparison
Two categorical cols + numeric measure stacked_bar Two dimensions → grouped comparison
Two numeric cols, no obvious category scatter Numeric × numeric → correlation
Single aggregatable numeric col kpi One number → headline metric
Anomaly-Driven Insight Tiles

Instead of generic summaries, insights are driven by actual anomaly signals from the profiler:

BEFORE (Generic)
"The data shows pipeline values across regions. AMERICAS has the highest total at $45M."
AFTER (Anomaly-Driven)
"EMEA pipeline dropped 23% QoQ — the only region declining. 68% of total pipeline is concentrated in just 3 accounts, creating significant risk."
Anomaly signals fed to LLM prompt:
  • Outliers: "Column 'Amount' has 3 outliers above $12M (IQR method)"
  • Trends: "Column 'CloseDate' shows decreasing values over time (r=-0.72)"
  • Concentration: "Top 3 of 45 accounts represent 68% of total Amount"
  • Period change: "Q1→Q2 delta: EMEA -23%, AMERICAS +8%, APAC +2%"
