Design Decisions

  • Headline insight — bold 16px title extracted from the first takeaway. Instant scanability.
  • Color-coded chart bars — green/yellow/red match the coverage thresholds. Dashed benchmark line adds context.
  • Blockquote-style additional insights — left border accent makes each takeaway scannable. Red border = risk callout.
  • Classification badges removed — SQL_QUERY / MODERATE are internal. Moved behind "Entities" pill.
  • Entity card hidden by default — accessible via pill for power users.
  • Confidence as dot + text — green glow dot is subtler and more glanceable.
